Doing Stuff with Web Things

What is a place holder ?

The 'placeholder thing' is a property specific to textarea's and input tags.

What is the var in JS ?

That var thing is what JavaScript programmers use to tell the computer to create a 'variable'.Variables are like boxes with labels on them.
Since you are the programmer you are the one who decides what label to put on the box, but just like in my kitchen, it is very wise to have a label that matches what the box has in it.

What is localStorage in JavaScript?

JavaScript's local storage uses what techies call 'key-value pairs.' That's fancy talk for 'description-stuff pairs.' If you've ever seen an old fashioned phone book you've seen this in action.
The 'key' would be the name of the person and the 'value' would be their address and phone number.

Key-values in localStorage?

The key-values this is similar to cache in php to store the details in the local system or in place we want. It is crucial for a site to remember the input the user had input and if it forgets the information it will be useless.

Mobile Master UX

Things to remember :

  • Mobile use has been increased rapidly and became more accessible and a necessity for many since the iPhone came out.This rapid increase in internet accessibility has an impact on how we make websites now.
  • People use mobile in mobility more than laptops they walk around with their mobile in hand rather than laptops.
  • Building the site from smallest screen like mobile with low bandwidth and specs in mind can help our website be more responsive in the biggers screens and more over the core functionalites remain when the website is scaled up to bigger screens.
  • Always optimise the content in the site using compressions of data like pictures, video or even the code itself.
  • Mobile is not a fad we will continue to have them for a long time.
  • Be there ,Be useful,Be quick
  • Keep it simple
  • Be emphatic with the user
  • Reachbility of screen realestate & Placement matters a lot
  • Speed matters we don't want the user to wait for content and put more cognitive load to the user.
  • Do not horde the user with advertisements use the ad space more wisely and in reserved manner because content matters.
  • Autofills or what ever makes the viewer experience fast in filling forms or searches is a plus for the website as they can have more ease of experience while navigating through their site or the purchases in a website.
